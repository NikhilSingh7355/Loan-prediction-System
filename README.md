# 🏡 Loan Prediction System

A Machine Learning based Loan Prediction System developed using Python, Streamlit, Pandas and Scikit-Learn.

## 📌 Project Overview

This application predicts whether a customer is eligible for a loan based on:

- Income
- CIBIL Score
- Loan Amount
- Employment Years

The model is built using the Decision Tree Classifier algorithm and provides loan approval predictions with high accuracy.

## 🚀 Features

- Loan Approval Prediction
- Interactive Streamlit User Interface
- Decision Tree Machine Learning Model
- Displays Model Accuracy
- Easy to Use

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-Learn

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/NikhilSingh7355/Loan-prediction-System.git
```

Go to project folder

```bash
cd Loan-prediction-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
streamlit run app.py
```
